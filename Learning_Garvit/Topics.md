## Learning Roadmap (Beginner → Project Ready)

1. JavaScript (Core)
   - async / await
   - Promises
   - ES Modules (import / export)
   - Map, Set

2. Node.js
   - What Node.js is
   - Event loop (high level)
   - HTTP module basics
   - process.env

3. Express
   - app.use()
   - Routes (GET, POST)
   - Middleware concept
   - Request → Response lifecycle

4. JWT Authentication
   - What JWT is
   - Token generation
   - Token verification
   - Auth middleware

5. WebSockets
   - HTTP vs WebSocket
   - Persistent connection
   - Client ↔ Server messaging
   - Event-based communication

6. Rooms / Channels
   - Why rooms are needed
   - Join / leave room
   - Broadcast vs global emit
   - One poll = one room

7. Redis
   - What Redis is
   - Key-value store
   - SET, GET, INCR
   - TTL
   - Pub/Sub basics

8. MySQL
   - Tables & relationships
   - Primary keys
   - Foreign keys
   - UNIQUE constraint (one vote per user)
   - Indexes (basic)

9. React Native Socket Flow
   - WebSocket connection
   - Sending events
   - Receiving updates
   - Cleanup on screen unmount

10. Project Implementation
    - Auth + Poll APIs
    - WebSocket integration
    - Live voting
    - One-vote-per-user logic
    - Redis + MySQL sync
